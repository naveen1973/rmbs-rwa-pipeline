RMBS RWA Pipeline

An end-to-end UK RMBS surveillance and securitisation regulatory-capital platform: ingest issuer loan-level tapes and investor reports, build a cloud data warehouse, compute Basel 3.1 / UK CRR securitisation RWA, and serve it to interactive dashboards.

▶ View the live dashboard

What it does

Ingest

Parses the European DataWarehouse AR-field loan-level tape (8,000+ loans), decodes coded fields, derives CPR/SMM/PSA, delinquency buckets, LTV and loss severity.

Warehouse

Loads curated facts and dimensions into a BigQuery warehouse (staging + marts), refreshed each month per deal.

RWA engine

Computes tranche attachment/detachment, thickness, SEC-ERBA risk weights, RWA and 8% capital — with period-on-period movement analysis.

Serve

A dependency-free interactive dashboard plus a Power BI model (star schema + DAX measures) for management information.
